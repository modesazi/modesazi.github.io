OpenRouter Image Lab

Nano Banana Studio

连接 OpenRouter 的 Nano Banana 图像模型，输入临时 API Key、上传参考图、填写提示词，然后直接在浏览器里拿到返回结果。

API Key 默认仅用于当前页面请求，不会自动写入本地。你可以手动点击“保存到本地”后再持久化到当前浏览器。
默认不自动保存；如果未手动保存，页面刷新后需要重新输入。
支持常见横屏比例；选择“自定义”后可手动输入 x:y。
支持拖拽或手动选择，可用于图像编辑、风格迁移、参考构图。

图像生成与图像编辑都可用
